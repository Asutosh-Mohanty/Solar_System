# 🌌 Solar System Simulation

An interactive and visually appealing **Solar System simulation** built using pure **HTML, CSS, and JavaScript**.

This project showcases planetary motion, animations, and dynamic space effects like twinkling stars, meteors, and asteroids — all running smoothly in the browser.

---

## 🚀 Features

- 🌞 Sun at the center with glow effects  
- 🪐 All 8 planets with realistic orbital motion  
- 🌍 Earth with a rotating Moon  
- 🌠 Twinkling stars background  
- ☄️ Random meteor animations  
- 🪨 Asteroids/satellites placed across orbits  
- 🎚️ Adjustable Earth orbital speed (interactive slider)  

---

## 🛠️ Tech Stack

- **HTML** – Structure  
- **CSS** – Animations, styling, orbit design  
- **JavaScript** – Dynamic elements (stars, meteors, controls)
